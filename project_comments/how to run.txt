# 주의사항
crawler.py, sentiment_analysis.py, wordcloud_gen.py, main.py, steam_comments.db (크롤링이 되어있다면)
가 모두 한 폴더 안에 있어야함

---준비물---
pip download list 에 있는 패키지 두 개 다운받는 명령어 실행
pip download list 다운 받은 후

---실행에 대해---
1. 경로가 project_comments로 되어있는지 확인, 폴더에 main.py가 있는지 확인
2. 터미널에 python main.py game_id
(game_id 에 해당하는 id를 넣어주면 됨.)
ex)
할로우나이트) 367520
스타듀밸리) 413150
유로트럭2) 227300
썸썸편의점) 1217390

---결과창---

크롤링 결과(db에 저장이 되었는가),
감정 분석 결과(긍정positive, 부정negative, 중립neutral),
워드클라우드 시각화(왼쪽부터 긍정,부정,중립)
(워드클라우드는 turtle처럼 파이썬 라이브러리 전용 프로그램으로 실행됨)
